# Resume PDF Setup

Place your resume PDF file in this `public/` folder with the name:
```
Samar_Singh_Resume.pdf
```

The file will be automatically served at:
```
/Samar_Singh_Resume.pdf
```

## How it works:
- Files in the `public/` folder are served at the root level
- The ResumeApp component loads the PDF via iframe
- Users can view and download the resume directly from the app panel

## Command:
Type `resume` in the terminal to open the resume viewer.
